# git 命令学习

## 初始化

```shell
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:uuRRx0/git-command.git
git push -u origin main
```

## 添加远程仓库到已有仓库

```shell
git remote add origin git@github.com:uuRRx0/git-command.git
git branch -M main
git push -u origin main
```
